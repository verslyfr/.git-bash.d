# AI Agent Workflow Guidelines

These rules apply to any AI agent or automated script interacting with this repository.

1. **Commit and Push Restrictions**: Do not commit and push changes unless explicitly directed to do so by the user.
2. **Documentation Evaluation**: Evaluate whether any instructions need to be added or updated in `AGENTS.md` and `README.md` before preparing a commit. Always keep the documentation in sync with new workflows or significant architectural changes.
